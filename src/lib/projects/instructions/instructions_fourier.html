1. Use Javascript text field to create vector arms:
    1. setSpeed(n), changes the speed of animation by n times. Default is 1.
    2. mainArm object has 2 functions push, pushA:
      1. push(length, speed) - Accepts two parameters that determine length and rotation speed of the vector arm.
      2. push(length, speed, angle) - Accepts three parameters, third being the value from the starting position of the unit circle, ranging from 0 to 2PI.
  2. Press "Animate" to run the script.
  3. Press "Toggle Arm" to set visibility of the vectors.
  4. Press "Save Canvas" to save the canvas as image.
