1. Press on the coloured cubes faces to build adjacent to the edge.
  2. Click Undo to detroy the last placed cube.
  3. The Second button selects the Mode of the visualization of the cubes:
    1. Fixed - disables all interactions with the cube.
    2. Move - allows the user to move the camera away and towards the cubes (W -> Up, S -> Down)
    3. Look - allows the uer to look around their camera position (WASD keys)
    4. Rotation - allows the user to rotate the shape along the x and y axis (WASD)
    5. Presentation mode colors the coloured block in the texture and disables the ability to build and disables the ability to accidentaly Undo build.
  4. The Third button changes between two types of projections of the objects:
    1. Perspective Projection - renders the cubes from one point of view like a real life camera.
    2. Orthographic Projection - renders the cubes using linear projection, with only the front view.
