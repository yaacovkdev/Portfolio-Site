After watching this awesome video that explains Fourier Series, made by 3Blue1Brown:

I decided to create a web app that takes a script and produces drawings from the added rotating vectors. Here are some really cool design patterns, you can try:

  1. mainArm.push(150,10);
    mainArm.push(150,-45);
  2. //This particular set of arms demonstrates the irrationality of PI
    mainArm.push(200,20);
    mainArm.push(200,Math.PI);

The Javascript text field provides the opportunity to write javascript that will be used in the sandbox environment that executes code within the sandbox of the canvas environment.
