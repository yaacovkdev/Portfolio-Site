After watching this awesome video that explains Fourier Series, made by 3Blue1Brown:

I decided to create a web app that takes a script and produces drawings from the rotating vectors. Here are some cool vectors you can observe:

  1. mainArm.push(150,10);
    mainArm.push(150,-25);
  2. //This particular set of arms demonstrates the irrationality of PI
    mainArm.push(200,10);
    mainArm.push(200,Math.pow(Math.E,Math.PI));
  3. mainArm.pushA(200,-2,0);
    mainArm.pushA(100,2,0);
    mainArm.pushA(50,30,Math.PI/2);
    mainArm.pushA(50,-30,Math.PI/2);

The Javascript text field sends the script to be executed in the sandbox environment that prevents malicious code from running.
